# 安心置地 - 安心學 (Peace and Art) 專屬動態簡報程式
您好, 我是 林水成
這是一個使用 Python 撰寫的互動式動態簡報程式，專為介紹「安心置地集團」及其精品建案「安心學」所設計。程式採用 **Pygame** 引擎驅動，提供平滑的淡入特效與專業的視覺呈現，適合用於展間數位看板或產品演示。

## 核心結論 (Core Summary)
本專案透過程式化的方式，將建案資訊轉化為具備「黑金」高質感的數位簡報。程式內嵌了安心學建案的五大核心優勢：**交通樞紐、日系建材、無暗房設計、品牌實力、優惠方案**。

---

## 視覺展示 (Visual Features)
- **專業配色**：採用深色背景搭配金、白、灰三色文字，呈現頂級建築與管理顧問風格。
- **淡入特效**：每頁投影片切換時均具備平滑的 Alpha 透明度過渡特效。
- **直覺操作**：支援滑鼠左鍵或鍵盤空白鍵快速切換。
